# ASM4 使用指南

## A Java bytecode engineering library

- 本文档是作者小傅哥从网上资料获取整理，方便学习使用。ASM 是一个 Java 字节码操控框架。它能被用来动态生成类或者增强既有类的功能。ASM 可以直接产生二进制 class 文件，也可以在类被加载入 Java 虚拟机之前动态改变类行为。如果本文能为您提供帮助，请给予支持(关注、点赞、分享)！
